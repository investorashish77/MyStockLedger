TROUBLESHOOTING
===============

Error: "No module named 'PyQt5'"
Fix: Re-run setup: python3 setup_agent.py

Error: "No module named 'database'"
Fix: Make sure __init__.py files are present in all folders

Error: Can't find main.py
Fix: Make sure main.py is in the root equity-tracker folder

Error: Database error
Fix: Delete data/equity_tracker.db and restart app

NEED HELP?
==========

1. Check that ALL files are copied
2. Verify __init__.py exists in each package folder
3. Make sure you're in the right directory
4. Try: python3 --version (should be 3.8+)
